# Pagekit Docker
PHP Docker for Pagekit

## What is included?

* Apache + PHP 7.1 + VOLUME ( thanks to https://hub.docker.com/_/php/ )
* Some PHP extensions enabled ( see below )
* Apache mod_rewrite enabled
